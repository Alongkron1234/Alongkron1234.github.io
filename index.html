AvocadoRipnessPredictor

This is an AI classification model designed to answer the question: 'What variety is this avocado, and how many days until it ripens?' The model was built using Fastai’s Datablock and Resnet34 architecture, achieving an accuracy of 82%. It has been deployed on Hugging Face Spaces.

A model of an autonomous driving system that detects road lanes from a camera or video input by distinguishing the road colors through computer processing. The data is then sent to a NodeMCU board for further processing before transmitting signals to the robot to execute the commands.
